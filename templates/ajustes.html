📡 API REST - FlaskCast

Para usar la API, debes tener una sesión activa en FlaskCast y tener la opción "Habilitar API" activada en Ajustes.

📤 Agregar Video

Sube un archivo de vídeo a una serie y temporada específica.

Petición

POST /api/videos/add
Content-Type: multipart/form-data

Parámetros

  • serie (obligatorio) — Nombre de la serie
  • temporada — Nombre de la temporada (obligatorio si la serie tiene subcarpetas, no uses si no las tiene)
  • archivo (obligatorio) — Archivo de vídeo

Ejemplo con curl (con temporada)

curl -X POST http://localhost:5000/api/videos/add \
  -b "session=TU_SESSION" \
  -F "serie=Mi Serie" \
  -F "temporada=Temporada 1" \
  -F "archivo=@/ruta/al/video.mp4"

Ejemplo (sin temporada)

curl -X POST http://localhost:5000/api/videos/add \
  -b "session=TU_SESSION" \
  -F "serie=Mi Serie" \
  -F "archivo=@/ruta/al/video.mp4"

📋 Listar Estructura

Obtén la estructura completa de todas las series o de una específica.

Petición

GET /api/videos              → Todas las series
GET /api/videos/Mi Serie     → Una serie específica

Respuesta (ejemplo)

{
  "status": "ok",
  "series": [
    {
      "nombre": "Mi Serie",
      "temporadas": {
        "Temporada 1": {
          "nombre": "Temporada 1",
          "capitulos": ["cap1.mp4", "cap2.mp4"]
        }
      }
    }
  ]
}

Ejemplo con curl

curl http://localhost:5000/api/videos \
  -b "session=TU_SESSION"

curl http://localhost:5000/api/videos/Mi%20Serie \
  -b "session=TU_SESSION"

🎬 Obtener Video

Devuelve el archivo de vídeo para reproducir o descargar.

Petición

GET /api/video/Mi Serie/Temporada 1/cap1.mp4

Ejemplo con curl

curl http://localhost:5000/api/video/Mi%20Serie/Temporada%201/cap1.mp4 \
  -b "session=TU_SESSION" \
  -o cap1.mp4

🗑️ Eliminar Video

Elimina un archivo de vídeo de una serie.

Petición

POST /api/videos/rm
Content-Type: application/json

Parámetros (JSON)

  • serie (obligatorio) — Nombre de la serie
  • filename (obligatorio) — Ruta del archivo dentro de la serie (ej: Temporada 1/video.mp4)

Ejemplo con curl

curl -X POST http://localhost:5000/api/videos/rm \
  -b "session=TU_SESSION" \
  -H "Content-Type: application/json" \
  -d '{"serie": "Mi Serie", "filename": "Temporada 1/video.mp4"}'
